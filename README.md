# pipelineAI

**pipelineAI** is an intelligent, LangGraph-based CI/CD automation platform designed to streamline infrastructure management. By leveraging agentic AI, it allows users to trigger and manage Azure pipelines using natural language prompts, reducing manual overhead and technical friction.



## 🚀 Key Features

* **Agentic Orchestration**: Utilizes `LangGraph` to manage complex, stateful multi-step workflows.
* **Natural Language Interface**: Translates human intent into actionable pipeline commands, removing the need for manual navigation of Azure portals.
* **Production-Grade Architecture**: Designed for reliability and scalability using a modern, reactive stack.

## 🛠 Tech Stack

* **Frontend**: Next.js (TypeScript) for a responsive, real-time user interface.
* **Backend**: Python (FastAPI) for high-performance service delivery and AI integration.
* **AI/Orchestration**: LangGraph for building stateful, multi-actor agentic pipelines.
